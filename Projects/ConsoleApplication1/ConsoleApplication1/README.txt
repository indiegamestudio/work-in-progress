CHALLENGE DESCRIPTION
////////////////////////////////////////////////////////////////////////////////

Write a simple game that asks the user to guess a random number betwee 1 and 100 
(inclusive). When the user guesses a number your program should tell them if they 
are above or below the number.  The game is over when they guess the number.


PERMITTED NAMESPACES
////////////////////////////////////////////////////////////////////////////////

System

////////////////////////////////////////////////////////////////////////////////

1 and 100 (inclusive).  Can you guess it?

> 75

You guessed the number in 4 tries!  Goodbye!

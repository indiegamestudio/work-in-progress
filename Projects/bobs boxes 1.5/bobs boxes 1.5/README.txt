CHALLENGE DESCRIPTION
--------------------------------------------------------------------------------------------
You have been hired by Bob's Boxes to write a program to calculate the cost of manufacturing
plastic boxes.  Each box includes a lid.


PERMITTED NAMESPACES
--------------------------------------------------------------------------------------------
System
System.IO


INPUT SPECIFICATION
--------------------------------------------------------------------------------------------
The file box_data.txt contains information on several types of boxes which are to be produced
in Bob's facility.  The first line of the data file contains the cost of plastic per cubic
centimetre in dollars.  Subsequent lines contain data for each type of box to be produced.  
For each line, the data fields are as follows:

field 1 -> box colour
field 2 -> inner length (cm)
field 3 -> inner width (cm)
field 4 -> inner height (cm)
field 5 -> plastic thickness (mm)
field 6 -> number of boxes


OUTPUT SPECIFICATION
--------------------------------------------------------------------------------------------
Output a file called costs.txt which exactly matches the file sample_costs.txt.
